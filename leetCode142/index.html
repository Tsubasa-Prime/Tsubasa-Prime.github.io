142. 环形链表 II

给定一个链表，返回链表开始入环的第一个节点。 如果链表无环，则返回 null。

为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。

说明：不允许修改给定的链表。

示例 1：

输入：head = [3,2,0,-4], pos = 1
输出：tail connects to node index 1
解释：链表中有一个环，其尾部连接到第二个节点。

解题思路
分别设定一个快指针，一个慢指针。快指针一次走两步，慢指针一次走一步。当快指针和慢指针相遇时，指向的是环的最后一个元素。让快指针指向第一个元素，与慢指针一同向前走一步，再次相遇时就是环的开始位置。

/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func detectCycle(head *ListNode) *ListNode {
    // 如果指针没有元素或只有一个元素，直接返回
    if head == nil || head.Next == nil {
        return nil
    }
    isCycle, slow := hasCycle(head)
    if !isCycle {
        return nil
    }
    // 相遇之后让快指针指向第一个元素
    fast := head
    // 再次相遇的点为环的起始位置
    for fast != slow {
        fast = fast.Next
        slow = slow.Next
    }
    return slow
}

// 判断是否有环
func hasCycle(head *ListNode) (bool, *ListNode) {
    fast := head
    slow := head
    // 快指针走两步，慢指针走一步，相遇点为环的最后一个元素
    for fast != nil && fast.Next != nil && slow != nil {
        slow = slow.Next
        fast = fast.Next.Next
        if fast == slow {
            return true, fast
        }
    }
    return false, nil
}
